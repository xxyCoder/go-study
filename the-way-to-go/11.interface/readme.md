# 接口与反射
- 类型不需要显式声明它实现了某个接口：接口被隐式实现，多个类型可以实现同一个接口
- 一个类型可以实现多个接口
- 接口类型包含一个实例的引用，该实例的类型实现了此接口
- 接口也可以嵌套一个或多个接口，相当于直接将这些接口的方法内嵌在外层接口一样
- 类型断言
  - v, ok := varI.(T) varI必须是接口变量
    - 如果成功那么v就是转换到T类型的值，ok是true
    - 如果失败那么v是T类型的零值，ok是false
  - v, ok := val.(method)
    - val是一个值，其有method方法
- type-switch 检查类型
